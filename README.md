# scheduli-bot
A bot that can schedule a set of persons on Discord.

## Usage
Install the program:
```sh
npm install
```

Start the program:
```sh
npm start
```

Test the program:
```sh
npm test
```

## Libraries
* Mocha and chai
* chrono
* discord.js
* dotenv
